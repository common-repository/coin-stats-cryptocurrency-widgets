Coin prices, charts, insights, coin converter and calculator widgets can be added as a great supplement to your site.

== Description ==

A set of ready-to-tailor widgets that can be incorporated on your site as an eye-catching display of the most useful cryptocurrency information, thus making a huge improvement on the user experience. Here is the available widgets’ list:

 -  [**Coin Price Widget Demo**](https://coinstats.app/en/widget/coin-price?type=large&wordpress=true)
 -  [**Coin List Widget Demo**](https://coinstats.app/en/widget/coin-list?type=large&wordpress=true)
 -  [**Coin Price Chart Widget Demo**](https://coinstats.app/en/widget/coin-price-chart?type=large&wordpress=true)
 -  [**Insight Widget Demo**](https://coinstats.app/en/widget/insight?type=howManyUsersHold&wordpress=true)
 -  [**Converter Widget Demo**](https://coinstats.app/en/widget/converter?type=large&wordpress=true)
 -  [**Coin Calculator Widget Demo**](https://coinstats.app/en/widget/coin-calc?type=invest&wordpress=true)
 -  [**Coin Price Marquee Widget**](https://coinstats.app/en/widget/price-marquee?wordpress=true)

All of the widgets can be customized by size, color and many other criterias to fit with the branding of your site. 

**Coin Price Widget**
A widget of a specific coin price.

**Coin List Widget**
A widget that displays a chosen list of coin prices.

**Coin Price Chart Widget**
A widget of the chart of a specific coin. The chart can be displayed starting from 24h till All time periods.

**Insights Widget**
This widget shows aggregated statistics of portfolios and coins tracked by users using CoinStats app.
There are 4 types of Insights:

1. *How many CoinStats users hold a specific coin.*
2. *Where do CoinStats users hold a specific coin, i.e. distribution between wallets and exchanges.*
3. *What is a coin dominance in CoinStats portfolio.*
4. *Today’s trades.*

**Converter Widget**
Allows to convert coins into different currencies.

**Coin Calculator Widget**
With this widget can be calculated how much profit you can make if you invest a given amount of coins and have the given price change.

**Coin Price Marquee Widget**
A widget that displays a list of top coin prices marquee.
